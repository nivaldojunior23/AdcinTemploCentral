# Adcin Templo Central — Site Institucional

> Site oficial da **ADCIN – Templo Central**, igreja localizada em Ananindeua – PA.  
> Desenvolvido para acolher visitantes, apresentar cultos e facilitar o acesso à comunidade.

---

## ✨ Funcionalidades

- **Dark Mode** automático (respeita o tema do sistema) com persistência via `localStorage`
- **Carrossel de imagens** animado em CSS puro na seção "Sobre Nós"
- **Grade de cultos** com animação scroll reveal encadeada
- **Menu mobile** hamburger em CSS puro, sem JavaScript
- **Mapa integrado** via Google Maps iframe
- **Scroll suave** entre seções
- Totalmente **responsivo** — mobile, tablet e desktop

---

## 🛠️ Tecnologias

| Tecnologia | Uso |
|---|---|
| HTML5 semântico | Estrutura da página |
| CSS3 puro | Todo o estilo, animações e responsividade |
| JavaScript vanilla | Dark mode toggle + scroll reveal |
| FontAwesome 6.4 | Ícones |
| Google Fonts (Inter) | Tipografia web |
| Netlify | Hospedagem + configuração de cache |

> Sem frameworks. Sem dependências de build. Abre direto no navegador.

## 📁 Estrutura

```
/
├── index.html          # Página principal
├── style.css           # Todo o CSS
├── netlify.toml        # Configurações de cache (Netlify)
├── AGENTS.md           # Guia para agentes de IA que trabalhem no projeto
└── public/
    ├── adcinvetor.svg      # Logo principal (branca/vazada)
    ├── Simplification.svg  # Logo alternativa (escura)
    ├── adcinvetor.png      # Logo PNG (apple-touch-icon)
    ├── site.webmanifest    # Web App Manifest
    └── *.jpg               # Fotos da igreja (hero + carrossel)
```

---

## 🚀 Como rodar localmente

Não há etapas de build. Basta clonar e abrir:

```bash
git clone https://github.com/seu-usuario/adcin-templo-central.git
cd adcin-templo-central
```

Abra o `index.html` diretamente no navegador, ou use uma extensão como **Live Server** no VS Code para ter recarregamento automático.
